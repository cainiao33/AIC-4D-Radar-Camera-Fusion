# SFA4D 可视化脚本详细文档

## 概述

本文档详细记录了 SFA4D 项目中所有可视化脚本的功能、特点、使用方法和参数说明。这些脚本主要用于可视化推理结果、点云数据和3D检测框。

## 脚本分类

### 1. 主要可视化脚本

#### 1.1 simple_visualize.py
**功能描述：** 简单的可视化脚本，生成基础的BEV（鸟瞰图）可视化
- **背景颜色：** 黑色背景
- **特点：** 
  - 使用 `np.zeros` 初始化BEV强度图
  - 添加白色边框和标题
  - 支持强度值归一化和裁剪
- **参数配置：**
  - `--dataset-dir`: 数据集根目录 (默认: `./DRadDataset`)
  - `--result-dir`: 推理结果目录 (默认: `./results`)
  - `--output-dir`: 输出目录 (默认: `./visualizations`)
  - `--sample-ids`: 样本ID列表 (默认: `[0, 1, 2, 3, 4]`)

#### 1.2 batch_visualize_ultra_aggressive.py
**功能描述：** 批量可视化超激进检测结果
- **背景颜色：** 黑色背景
- **特点：**
  - 使用 `np.zeros` 初始化BEV强度图
  - 专门用于可视化超激进检测结果
  - 显示检测统计信息（类别计数）
  - 支持处理样本数量限制
- **参数配置：**
  - `--dataset-dir`: 数据集根目录 (默认: `./DRadDataset`)
  - `--result-dir`: 推理结果目录 (默认: `./ultra_aggressive_results`)
  - `--output-dir`: 输出目录 (默认: `./ultra_aggressive_visualizations`)
  - `--num-samples`: 处理样本数量 (默认: 全部)

#### 1.3 batch_visualize_3d_boxes_fixed.py ⭐
**功能描述：** 修正版3D框批量可视化脚本
- **背景颜色：** 白色背景 ⭐
- **特点：**
  - 使用 `np.full((bev_height, bev_width, 3), 255, dtype=np.uint8)` 创建白色背景
  - 基于高度（z坐标）的彩色点云可视化
  - 支持180度旋转BEV
  - 修正版本，包含检测框
- **参数配置：**
  - `--dataset-dir`: 数据集根目录 (默认: `./DRadDataset`)
  - `--result-dir`: 推理结果目录 (默认: `./results`)
  - `--output-dir`: 输出目录 (默认: `./visualizations_3d_boxes_fixed`)
  - `--num-samples`: 处理样本数量 (默认: 全部)
  - `--rotate-bev`: 生成旋转BEV可视化 (标志参数)

#### 1.4 batch_visualize_3d_boxes_final.py ⭐
**功能描述：** 最终版3D框批量可视化脚本
- **背景颜色：** 白色背景 ⭐
- **特点：**
  - 使用 `np.full((bev_height, bev_width, 3), 255, dtype=np.uint8)` 创建白色背景
  - 修正的BEV纵横比和坐标对齐
  - 使用蓝-绿-红渐变的彩色点云
  - BEV旋转以匹配相机视角
- **参数配置：**
  - `--dataset-dir`: 数据集根目录 (默认: `./DRadDataset`)
  - `--result-dir`: 推理结果目录 (默认: `./ultra_aggressive_results`)
  - `--output-dir`: 输出目录 (默认: `./visualizations_final`)
  - `--num-samples`: 处理样本数量 (默认: 全部)

#### 1.5 batch_visualize_3d_boxes.py
**功能描述：** 基础3D框批量可视化脚本
- **背景颜色：** 黑色背景
- **特点：**
  - 使用 `np.zeros` 初始化BEV图
  - 基础版本的3D框可视化
  - 支持正常和旋转BEV生成
- **参数配置：**
  - `--dataset-dir`: 数据集根目录 (默认: `./DRadDataset`)
  - `--result-dir`: 推理结果目录 (默认: `./ultra_aggressive_results`)
  - `--output-dir`: 输出目录 (默认: `./visualizations_3d_boxes_batch`)
  - `--num-samples`: 处理样本数量 (默认: 全部)
  - `--rotate-bev`: 生成旋转BEV可视化 (标志参数)

#### 1.6 visualize_with_3d_boxes.py
**功能描述：** 带3D框的可视化脚本
- **背景颜色：** 黑色背景
- **特点：**
  - 将3D框投影到2D图像上
  - 检查BEV旋转
  - 添加白色边框用于组合图像
- **参数配置：**
  - `--dataset-dir`: 数据集根目录 (默认: `./DRadDataset`)
  - `--result-dir`: 推理结果目录 (默认: `./ultra_aggressive_results`)
  - `--output-dir`: 输出目录 (默认: `./visualizations_3d_boxes`)
  - `--sample-ids`: 要可视化的样本ID (默认: `[0, 1, 2, 10, 625]`)
  - `--rotate-bev`: 将BEV旋转180度 (标志参数)

### 2. 演示脚本

#### 2.1 demo_front.py
**功能描述：** 前视图演示脚本
- **特点：**
  - 仅演示前视图
  - 包含数据集下载功能
  - 模型创建和加载
  - 设备推理设置

#### 2.2 demo_2_sides.py
**功能描述：** 双视图演示脚本
- **特点：**
  - 演示前视图和后视图
  - 包含数据集下载功能
  - 模型创建和加载
  - 设备推理设置

### 3. 视频创建脚本

#### 3.1 create_video_from_images.py
**功能描述：** 从图像创建视频
- **特点：**
  - 使用OpenCV从PNG图像目录生成视频
  - 可配置FPS（默认24，示例中设为12）
  - 支持自定义输出路径
- **使用示例：**
  ```python
  create_video(
      image_dir="./visualizations",
      output_path="./visualization_video.mp4",
      fps=12
  )
  ```

### 4. 调试和分析脚本

#### 4.1 visualize_inference_results.py
**功能描述：** 可视化推理结果并验证雷达设备加载
- **特点：**
  - 从 `.bin` 文件加载点云
  - 检查点云结构和设备ID
  - 验证雷达设备加载

#### 4.2 analyze_dataset_distribution.py
**功能描述：** 分析数据集分布
- **特点：**
  - 设置自动文本颜色为白色
  - 数据集统计分析

### 5. 工具脚本

#### 5.1 utils/demo_utils.py
**功能描述：** 演示工具函数
- **特点：**
  - 定义白色颜色变量
  - 提供演示相关的工具函数

## 白色背景BEV脚本总结

根据分析，以下脚本生成**白色背景**的BEV图像：

1. **batch_visualize_3d_boxes_fixed.py** ⭐
   - 修正版本，支持旋转BEV
   - 基于高度的彩色点云

2. **batch_visualize_3d_boxes_final.py** ⭐
   - 最终版本，修正纵横比
   - 蓝-绿-红渐变彩色点云

其他脚本（`simple_visualize.py`、`batch_visualize_ultra_aggressive.py`、`batch_visualize_3d_boxes.py`）使用**黑色背景**。

## 使用建议

1. **对于标准可视化：** 使用 `simple_visualize.py`
2. **对于白色背景BEV：** 使用 `batch_visualize_3d_boxes_final.py`（推荐）或 `batch_visualize_3d_boxes_fixed.py`
3. **对于超激进检测结果：** 使用 `batch_visualize_ultra_aggressive.py`
4. **对于视频生成：** 使用 `create_video_from_images.py`

## 快速使用指南

### 基本使用命令

```bash
# 使用默认参数运行简单可视化
python simple_visualize.py

# 使用白色背景BEV（推荐用于展示）
python batch_visualize_3d_boxes_final.py

# 自定义输出目录
python batch_visualize_3d_boxes_final.py --output-dir "D:\my_visualizations"

# 只处理前10个样本
python batch_visualize_ultra_aggressive.py --num-samples 10

# 生成旋转BEV
python batch_visualize_3d_boxes_fixed.py --rotate-bev

# 可视化特定样本
python visualize_with_3d_boxes.py --sample-ids 0 1 2 5 10
```

### 路径配置说明

所有脚本的默认路径配置：
- **数据集目录：** `./DRadDataset`
- **结果目录：** `./results` 或 `./ultra_aggressive_results`
- **输出目录：** 各脚本有不同的默认输出目录

### 输出文件说明

- **图像格式：** PNG
- **命名规则：** `{sample_id:06d}_vis.png` 或类似格式
- **组合图像：** 包含原始图像、BEV图和检测结果的组合视图

## 注意事项

- 所有脚本都支持自定义输入输出路径
- 白色背景脚本提供更好的打印和展示效果
- 部分脚本支持样本数量限制以便快速测试
- 建议在使用前检查路径配置是否正确
- 确保数据集和推理结果文件存在于指定路径
- 输出目录会自动创建，无需手动创建
