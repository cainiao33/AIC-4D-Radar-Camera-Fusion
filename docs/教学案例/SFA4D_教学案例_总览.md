# SFA4D 算法竞赛教学案例总览

## 🎓 案例定位

本教学案例源自**2025年AIC算法竞赛**获奖作品，完整呈现了一个从问题定义、算法设计、模型实现到工程部署的**端到端深度学习项目全生命周期**。适用于以下教学场景：

| 教学场景 | 适用课程 | 核心知识点 |
|----------|----------|------------|
| **深度学习入门** | 人工智能导论、机器学习 | 卷积神经网络、损失函数设计、训练技巧 |
| **计算机视觉进阶** | 计算机视觉、模式识别 | 3D目标检测、BEV投影、Anchor-Free检测 |
| **自动驾驶感知** | 自动驾驶技术、智能感知 | 毫米波雷达、多传感器融合、实时推理 |
| **工程实践** | 软件工程、系统设计 | 模型部署、ONNX导出、量化压缩、MLOps |

## 📊 案例亮点

### 🏆 竞赛成绩
- **🏅 国赛二等奖**：2025年AIC算法竞赛全国总决赛
- **📊 全国排名**: **10/85**（前12%），较基线提升71.4%
- **核心指标**: 75% mAP@0.5，110.73 FPS推理速度
- **模型压缩**: 73.3%压缩率，48.66MB → 13MB

### 🔬 技术创新（3项首创）
1. **8D→4D智能维度映射**：首创基于信噪比强度(P)的毫米波雷达数据降维方案
2. **KFPN自适应特征融合**：基于Softmax注意力机制的多尺度特征金字塔
3. **跨类别NMS后处理**：突破传统同类NMS局限，消除跨类重复检测

### 🛠️ 工程完整性
- 完整训练流水线（300 epochs，163轮最佳模型）
- 三阶段部署方案（PyTorch → ONNX → 量化）
- 1034样本全量验证，100%推理成功率

## 🎯 教学目标

### 知识目标
- 理解3D目标检测的基本范式（Anchor-Based vs Anchor-Free）
- 掌握BEV（鸟瞰图）投影的原理与实现
- 理解特征金字塔网络（FPN）的多尺度融合机制
- 掌握Focal Loss解决类别不平衡的数学原理

### 能力目标
- 能够设计完整的深度学习训练流水线
- 能够进行系统的消融实验和性能分析
- 能够将研究模型转化为生产可部署方案
- 能够撰写规范的技术报告和竞赛方案

### 素养目标
- 培养"问题定义→方案设计→实验验证→工程落地"的系统思维
- 建立传感器选型与算法适配的工程意识
- 理解学术研究与工业落地的差异与桥梁

## 📖 建议教学路径

### 路径一：快速入门（4课时）
1. 阅读[总览文档]（本文）→ 建立全局认知
2. 阅读[技术报告] → 理解完整方案
3. 阅读[实验结果] → 掌握评估方法
4. 动手跑通推理Demo → 获得直观感受

### 路径二：深度研学（8课时）
1. 第1-2课时：数据层深度剖析（8D→4D映射 + BEV投影）
2. 第3-4课时：网络架构逐层解析（ResNet → FPN → KFPN → Head）
3. 第5-6课时：训练策略与损失函数设计
4. 第7-8课时：消融实验复现与部署实践

### 路径三：项目实战（16课时）
- 在路径二基础上增加：
  - 数据集构建与标注实践
  - 超参数调优实验
  - 模型压缩与量化实战
  - 完整竞赛方案撰写
