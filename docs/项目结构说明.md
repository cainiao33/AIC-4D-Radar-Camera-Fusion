## 📦 模型目录详解

### 🔥 训练模型 (checkpoints/)
- **fpn_resnet_18/**: 原始SFA3D模型
  - `fpn_resnet_18_epoch_300.pth`: 300轮训练的基础模型

- **163_model.pth**: 163轮增强训练模型
  - 优化了8D雷达数据处理
  - 使用超激进NMS参数
  - 性能更优的检测效果

- **163_model_quantized.pth**: 量化压缩模型
  - 使用PyTorch官方量化
  - 压缩率73.3% (48.66MB → 13MB)
  - 适合内存受限环境

### 🌐 部署模型 (onnx_models/)
- **sfa3d_163_fp32.onnx**: ONNX格式部署模型
  - 跨平台兼容
  - 支持ONNX Runtime推理
  - 文件大小: 48.57MB
  - 推理速度: 5.48 FPS

## 📊 数据集目录详解

### 🎯 KITTI数据集 (dataset/kitti/)
- **原始数据**: LiDAR点云 + 图像 + 标注
- **用途**: 原始SFA3D训练和测试

### 📡 8D雷达数据集 (DRadDataset/)
- **ImageSets/**: 数据集索引文件
  - `test.txt`: 测试集索引
  - `train.txt`: 训练集索引
  - `val.txt`: 验证集索引 (1034个样本)

- **training/**: 训练数据目录
  - `image_2/`: 左彩色相机图像
  - `calib/`: 相机校准矩阵
  - `label_2/`: 对象标注文件
  - `velodyne/`: 3D点云数据

## 🛠️ 核心代码目录详解

### ⚙️ 配置模块 (config/)
- **kitti_config.py**: KITTI数据集配置
  - 类别定义 (Car, Cyclist, Truck)
  - 边界框参数设置
  - BEV投影参数

- **train_config.py**: 训练配置
  - 学习率设置
  - 批次大小配置
  - 优化器参数

### 📥 数据处理模块 (data_process/)
- **kitti_dataloader.py**: 数据加载器
  - 批量数据加载
  - 数据预处理流水线

- **kitti_dataset.py**: 数据集类
  - 数据集接口定义
  - 数据增强方法

- **kitti_data_utils.py**: 数据处理工具
  - 点云处理函数
  - BEV投影算法
  - 坐标变换工具

### 🧠 模型定义模块 (models/)
- **fpn_resnet.py**: 主干网络
  - ResNet-18 + FPN架构
  - 特征金字塔网络实现
  - KFPN特征融合

- **resnet.py**: ResNet基础网络
  - 残差块实现
  - 特征提取器

- **model_utils.py**: 模型工具函数
  - 模型初始化工具
  - 辅助函数

### 🔧 工具函数模块 (utils/)
- **evaluation_utils.py**: 评估工具 ⭐ 增强版
  - 原始PyTorch评估函数
  - **新增NumPy版本函数** (用于ONNX):
    - `decode_numpy()`: NumPy检测结果解码
    - `post_processing_numpy()`: NumPy后处理
    - `_nms_numpy()`: NumPy热图NMS
    - `_topk_numpy()`: NumPy TopK操作
  - 跨类别NMS优化

- **misc.py**: 通用工具函数
  - 文件操作工具
  - 数据转换工具

- **visualization_utils.py**: 可视化工具
  - 3D边界框绘制
  - 检测结果可视化

- **train_utils.py**: 训练工具
  - 损失函数计算
  - 梯度计算工具

## 🚀 脚本文件详解

### 🔥 增强版核心脚本 (推荐使用)

#### 推理脚本
- **run_163_working.py**: PyTorch优化推理 ⭐ 主要脚本
  - 163轮模型推理
  - 超激进NMS (peak_thresh=0.25, nms_thresh=0.2)
  - 完整验证集处理 (1034样本)
  - 性能统计和分析

#### ONNX部署脚本
- **export_to_onnx.py**: 模型ONNX导出
  - PyTorch → ONNX转换
  - 模型结构验证
  - 输入输出分析

- **run_onnx_inference.py**: ONNX Runtime推理
  - 跨平台推理执行
  - NumPy后处理支持
  - KITTI格式结果保存

#### 模型优化脚本
- **quantize_model_163.py**: PyTorch官方量化
  - 静态量化 (73.3%压缩)
  - 动态量化选项
  - 独立量化配置

#### 评估脚本
- **simple_onnx_evaluation.py**: ONNX性能统计
  - 检测结果统计分析
  - 类别检测率计算
  - 性能预估报告

- **kitti_evaluation_163.py**: KITTI标准评估
  - 遵循KITTI评测标准
  - mAP计算
  - 详细评估报告

### 📊 原始脚本 (参考使用)
- **test.py**: 原始推理脚本
- **train.py**: 原始训练脚本
- **demo_2_sides.py**: 双视角演示
- **demo_front.py**: 前视角演示

### ⚙️ 配置脚本
- **quantize_config.py**: 独立量化配置
  - 量化参数设置
  - 不影响训练配置

## 🎯 使用建议

### 🔬 研究开发
1. 使用 **run_163_working.py** 进行模型验证
2. 参考 **Technical_details.md** 了解技术细节
3. 查看 **项目总结.md** 了解整体架构

### 🚀 生产部署
1. 使用 **export_to_onnx.py** 导出ONNX模型
2. 使用 **run_onnx_inference.py** 进行跨平台推理
3. 参考 **ONNX_inference_summary.md** 部署指南

### 📱 资源受限环境
1. 使用 **quantize_model_163.py** 进行模型压缩
2. 选择量化模型减少内存占用
3. 调整推理批次大小

### 📊 性能评估
1. 使用 **simple_onnx_evaluation.py** 统计分析
2. 使用 **kitti_evaluation_163.py** 标准评估
3. 对比不同配置的性能表现
