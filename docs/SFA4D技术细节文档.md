# SFA4D 技术细节文档

## 1. 网络架构

- **ResNet-based Keypoint Feature Pyramid Network (KFPN)** - 基于[RTM3D论文](https://arxiv.org/pdf/2001.03343.pdf)
- **输入**:
  - 鸟瞰图(BEV)作为输入
  - BEV图由点云的高度、强度和密度编码
  - 输入尺寸: `(H, W, 3)` = `(608, 608, 3)`

- **输出**:
  - 中心点热力图: `(H/S, W/S, C)` 其中 `S=4` (下采样比), `C=3` (类别数)
  - 中心偏移: `(H/S, W/S, 2)`
  - 方向角(yaw): `(H/S, W/S, 2)` - 估计虚部和实部 (`sin(yaw)` 和 `cos(yaw)`)
  - 尺寸(h, w, l): `(H/S, W/S, 3)`
  - z坐标: `(H/S, W/S, 1)`

- **目标**: **7自由度** (7-DOF) 物体: `(cx, cy, cz, l, w, h, θ)`
  - `cx, cy, cz`: 中心坐标
  - `l, w, h`: 边界框的长宽高
  - `θ`: 边界框的偏航角(弧度)

- **检测类别**: Car, Cyclist, Truck

## 2. 损失函数

- **中心点热力图**: 使用 `focal loss`
- **方向角(yaw)**: 虚部和实部直接使用 `l1_loss` 回归
- **z坐标和3维尺寸**: 使用[Libra R-CNN](https://arxiv.org/pdf/1904.02701.pdf)提出的 `balanced l1 loss`

## 3. 训练细节

- 损失权重: 均匀权重 `=1.0` (所有组件)
- 训练轮数: 300
- 学习率调度: [`cosine`](https://arxiv.org/pdf/1812.01187.pdf), 初始学习率: 0.001
- 批次大小: `16` (单张GTX 4060Ti)

## 4. 推理

- 在中心热力图上应用 `3×3` 最大池化操作
- 保留中心置信度 > 0.2 的前 `50` 个预测
- 方向角(yaw) = `arctan`(虚部/实部)
