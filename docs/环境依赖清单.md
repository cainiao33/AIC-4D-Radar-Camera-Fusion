## 🚀 **推荐安装方法** (现代版本)

### 1. 创建虚拟环境
```bash
# 使用conda (推荐)
conda create -n sfa3d python=3.8
conda activate sfa3d

# 或使用venv
python -m venv sfa3d_env
# Windows
sfa3d_env\Scripts\activate
# Linux/macOS
source sfa3d_env/bin/activate
```

### 2. 安装PyTorch (CUDA 11.8版本)
```bash
# 推荐: PyTorch 2.0 + CUDA 11.8 (已验证稳定)
pip install torch==2.0.0+cu118 torchvision==0.15.1+cu118 torchaudio==2.0.1+cu118 --index-url https://download.pytorch.org/whl/cu118

# 或者使用conda安装
conda install pytorch==2.0.0 torchvision==0.15.1 torchaudio==2.0.1 pytorch-cuda=11.8 -c pytorch -c nvidia
```

### 3. 安装稀疏卷积库
```bash
# 安装spconv (3D点云处理必需)
pip install spconv-cu118==2.3.6
```

### 4. 安装其他依赖
```bash
pip install -r requirements.txt
```

## 常见问题

### 1. CUDA相关问题
```bash
# 检查CUDA版本
nvcc --version
nvidia-smi

# 重新安装PyTorch CUDA版本
pip uninstall torch torchvision
pip install torch==1.5.0+cu101 torchvision==0.6.0+cu101 -f https://download.pytorch.org/whl/torch_stable.html
```

### 2. OpenCV安装问题
```bash
# 如果opencv-python安装失败
pip install opencv-python-headless==4.2.0.34
```

### 3. 内存不足问题
- 减少batch_size (默认8，可降至4或2)
- 使用梯度累积
- 启用混合精度训练

### 4. 版本冲突问题
```bash
# 清理环境重新安装
pip freeze > installed_packages.txt
pip uninstall -r installed_packages.txt -y
pip install -r requirements.txt
```

## 性能优化建议

### 1. GPU优化
- 使用CUDA 10.1/10.2 + cuDNN 7.6.5
- 启用Tensor Core (RTX系列GPU)
- 设置合适的batch_size

### 2. 内存优化
- 使用torch.cuda.empty_cache()清理GPU内存
- 启用数据并行处理
- 优化数据加载器num_workers

### 3. 训练优化
- 使用混合精度训练 (AMP)
- 启用梯度检查点
- 使用分布式训练
