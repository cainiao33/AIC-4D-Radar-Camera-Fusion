# SFA4D 可视化指令文档

## 概述

本文档包含SFA4D项目的可视化指令，用于生成点云BEV图像和3D检测框的可视化结果。

## 环境要求

- Python 3.8+
- PyTorch 2.0.0+cu118
- CUDA 11.8+
- conda环境：sfa3d

## 激活环境

```bash
source activate sfa3d
```

## 白底可视化脚本（推荐）

### 核心可视化指令

```bash
source activate sfa3d && python sfa/batch_visualize_3d_boxes_final.py --dataset-dir ./DRadDataset --result-dir ./results/sfa4d_all --output-dir ./visualizations_final_all
```

**参数说明：**
- `--dataset-dir`: 数据集根目录
- `--result-dir`: 推理结果目录
- `--output-dir`: 可视化图像输出目录

### 测试可视化指令（10个样本）

```bash
source activate sfa3d && python sfa/batch_visualize_3d_boxes_final.py --dataset-dir ./DRadDataset --result-dir ./results/sfa4d_all --output-dir ./visualizations_test --num-samples 10
```

## 两个白底脚本对比

### 1. 最终版本（推荐） - `batch_visualize_3d_boxes_final.py`

```bash
source activate sfa3d && python sfa/batch_visualize_3d_boxes_final.py --dataset-dir ./DRadDataset --result-dir ./results/sfa4d_all --output-dir ./visualizations_final_all
```

**特点：**
- ✅ **白色背景** - 适合展示和打印
- ✅ **蓝-绿-红渐变彩色点云** - 基于强度值的渐变
- ✅ **标准坐标系** - 清晰的坐标说明
- ✅ **修正纵横比** - 准确的比例显示
- 🚀 **高处理速度**: 37.93 samples/s

**输出格式：**
- 文件名：`000000_final.png` 到 `001381_final.png`
- 尺寸：标准的BEV分辨率

### 2. 修正版本 - `batch_visualize_3d_boxes_fixed.py`

```bash
source activate sfa3d && python sfa/batch_visualize_3d_boxes_fixed.py --dataset-dir ./DRadDataset --result-dir ./results/sfa4d_all --output-dir ./visualizations_fixed_all --rotate-bev
```

**特点：**
- ✅ **白色背景** - 适合展示和打印
- ✅ **基于高度的彩色点云** - 不同高度不同颜色
- ✅ **支持旋转BEV** - `--rotate-bev` 参数
- ✅ **修正版本** - 修复了坐标系统问题
- 📊 **处理速度**: 19.84 samples/s

**可选参数：**
- `--rotate-bev`: 生成旋转的BEV视图

### 图像内容

每个可视化图像包含：
- **白色背景**: 清晰的显示效果
- **BEV点云**: 根据强度或高度着色的点云数据
- **检测框**: 3D边界框的BEV投影
- **坐标系统**: 标准的鸟瞰图坐标系

### 检测框颜色

- **Car**: 绿色边框 `(0, 255, 0)`
- **Pedestrian**: 蓝色边框 `(255, 0, 0)`
- **Cyclist**: 红色边框 `(0, 0, 255)`
- **其他类**: 青色边框 `(255, 255, 0)`

## 坐标系统说明

### BEV坐标系（最终版本）

```
BEV Coordinate System:
- Top of BEV = Forward (Y+)
- Left of BEV = Left (X-)
- Right of BEV = Right (X+)
- Front vehicle appears at top of BEV
- Left-front vehicle appears at top-left of BEV
```

### 点云数据格式

- **8D输入**: [x, y, z, D, P, R, A, E]
- **读取维度**: [0, 1, 2, 4] → [x, y, z, intensity]
- **显示**: 白色背景BEV图像 + 彩色点云 + 检测框
